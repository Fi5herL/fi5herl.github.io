Chrome Extension Privacy Policy

Flex Email Generator (Excel)

This page provides the privacy policy for the Flex Email Generator (Excel) Chrome extension.

Last updated: 2026-05-12

Privacy Policy (English)

Flex Email Generator (Excel) is a single-purpose Chrome extension that helps users generate Flex Project emails on portal.ul.com from local .xlsx templates synchronized through OneDrive. The extension does not collect personal data and does not send data to external servers.

1. What the extension does

The extension adds an email generation interface to the portal.ul.com project details page so the user can create email content from locally available Excel templates. Its single purpose is to support this email generation workflow.

2. Permissions requested and why

storage Stores local user settings and preferences in the browser only. These settings are never sent to any server.
clipboardWrite Copies generated email content to the clipboard for user convenience so it can be pasted into an email client.
activeTab Interacts with the active portal.ul.com tab to inject the email generation UI when the user is on the relevant page.
Host permission Access to https://portal.ul.com/* is used only to inject content scripts, add the email generation button, and read project metadata needed to prepare the email. No data is transmitted externally.

3. Data collection and transmission

No personal data is collected or transmitted. All data handled by the extension stays local in the browser and is used only to generate the requested email content for the user.

4. No remote code

All JavaScript used by the extension is bundled within the extension package. No remote code is loaded at runtime.

5. Third-party sharing

The extension does not sell, share, or transfer user data to third parties.

6. Contact information

For questions or updates, please refer to the GitHub repository: https://github.com/Fi5herL/ULS-EmailGenerator

隱私權政策（繁體中文）

Flex Email Generator (Excel) 是一個單一用途的 Chrome 擴充功能， 用於在 portal.ul.com 上，依據透過 OneDrive 同步的本機 .xlsx 範本產生 Flex Project 電子郵件內容。此擴充功能不蒐集個人資料， 也不會將資料傳送到任何外部伺服器。

1. 擴充功能用途

此擴充功能會在 portal.ul.com 的專案詳細頁面加入郵件產生介面， 讓使用者能根據本機可用的 Excel 範本建立電子郵件內容。其單一用途即為支援此郵件產生流程。

2. 申請的權限與用途說明

storage 僅用於在瀏覽器本機儲存使用者設定與偏好，不會傳送到任何伺服器。
clipboardWrite 用於將產生的郵件內容複製到剪貼簿，方便使用者貼到郵件軟體中。
activeTab 用於在使用者目前開啟的 portal.ul.com 分頁上互動， 並在相關頁面插入郵件產生介面。
網站權限 https://portal.ul.com/* 的權限僅用於注入內容腳本、 新增郵件產生按鈕，以及讀取準備郵件所需的專案中繼資料；不會對外傳輸資料。

3. 資料蒐集與傳輸

本擴充功能不蒐集或傳輸任何個人資料。所有資料都保留在瀏覽器本機， 僅用於替使用者產生所需的郵件內容。

4. 不使用遠端程式碼

擴充功能使用的所有 JavaScript 都已打包在擴充功能套件內， 執行時不會載入任何遠端程式碼。

5. 不與第三方分享資料

本擴充功能不會出售、分享或移轉使用者資料給任何第三方。

6. 聯絡資訊

如有問題或需更新資訊，請參考 GitHub repository： https://github.com/Fi5herL/ULS-EmailGenerator
